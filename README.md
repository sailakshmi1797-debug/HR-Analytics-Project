# HR-Analytics-Project
An end-to-end HR Analytics project using SQL to transform raw data and Power BI to visualize key attrition drivers like Overtime and Age Group.

🛠️ Tech Stack

1.Database: PostgreSQL/MySQL (SQL)

2.Visualization: Power BI

3.Transformation: SQL Views & Power Query

4.Analytics: DAX (Data Analysis Expressions)

📊 Project Workflow

1.Data Extraction: Imported raw HR data into a SQL database.

2.SQL Transformation: Created a VIEW to clean data, handle nulls, and bucket Age into logical groups for better visualization.

3.DAX Implementation: Developed measures for Attrition Rate, Total Headcount, and Average Salary.

4.UI/UX Design: Built a dark-themed dashboard focused on key drivers: Overtime, Age, and Department.

💡 Key Findings

1.Overtime Impact: Employees working overtime are significantly more likely to leave the company (Attrition rate of ~54% within the 'Yes' overtime group).

2.Demographic Risk: The "Under 30" age group shows the highest attrition frequency.

3.Departmental Trends: Identified the Sales department as a high-turnover area compared to Research & Development.
